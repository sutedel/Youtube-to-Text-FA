# برنامه تبدیل فایل صوتی به متن
# Audio File to Text Converter

برنامه‌ای ساده برای تبدیل فایل‌های صوتی به متن با پشتیبانی از زبان فارسی.

A simple program to convert audio files to text with Persian language support.

## 🎯 ویژگی‌ها / Features

- ✅ تبدیل فایل‌های صوتی به متن
- ✅ پشتیبانی از زبان فارسی (fa-IR)
- ✅ پشتیبانی از زبان انگلیسی (en-US) به عنوان جایگزین
- ✅ ذخیره خروجی در فرمت TXT و JSON
- ✅ رابط کاربری ساده و فارسی

## 📋 نیازمندی‌ها / Requirements

- Python 3.7+
- SpeechRecognition
- PyAudio
- اتصال اینترنت (برای Google Speech Recognition)

## 🚀 نصب / Installation

### 1. نصب Python Dependencies
```bash
pip install SpeechRecognition PyAudio
```

### 2. دانلود فایل‌های برنامه
- `final_mp3_to_text.py` - برنامه اصلی
- `run_mp3.bat` - فایل اجرا برای Windows

## 📁 فرمت‌های پشتیبانی شده / Supported Formats

**فقط WAV** - برای بهترین نتیجه

### تبدیل فرمت‌های دیگر به WAV:

#### 🎵 Audacity (رایگان)
1. دانلود از: https://www.audacityteam.org/
2. فایل صوتی را باز کنید
3. File → Export → Export as WAV

#### 🎵 VLC Media Player
1. فایل صوتی را باز کنید
2. Media → Convert/Save
3. فرمت خروجی را WAV انتخاب کنید

#### 🌐 آنلاین کانورتر
- https://convertio.co/mp3-wav/
- https://www.onlinevideoconverter.com/

## 🎮 استفاده / Usage

### روش 1: اجرای مستقیم
```bash
python final_mp3_to_text.py
```

### روش 2: استفاده از فایل batch (Windows)
```bash
run_mp3.bat
```

### مراحل استفاده:
1. برنامه را اجرا کنید
2. مسیر فایل WAV را وارد کنید
3. منتظر بمانید تا پردازش کامل شود
4. فایل‌های خروجی را بررسی کنید

## 📄 فایل‌های خروجی / Output Files

### transcript.txt
فایل متنی ساده حاوی متن تشخیص داده شده

### transcript.json
فایل JSON با اطلاعات کامل:
```json
{
  "input_file": "path/to/audio.wav",
  "transcript": "متن تشخیص داده شده",
  "method": "Google Speech Recognition"
}
```

## 🔧 عیب‌یابی / Troubleshooting

### مشکل: فایل MP3 پشتیبانی نمی‌شود
**راه‌حل:** فایل را به WAV تبدیل کنید

### مشکل: گفتار تشخیص داده نمی‌شود
**راه‌حل‌ها:**
- کیفیت صدا را بررسی کنید
- فایل باید گفتار واضح داشته باشد
- اتصال اینترنت را بررسی کنید

### مشکل: خطا در اتصال
**راه‌حل:** اتصال اینترنت خود را بررسی کنید

## 🎯 نکات مهم / Important Notes

- **کیفیت صدا:** هر چه کیفیت صدا بهتر باشد، تشخیص دقیق‌تر است
- **زبان:** برنامه ابتدا فارسی را امتحان می‌کند، سپس انگلیسی
- **اندازه فایل:** فایل‌های بزرگ ممکن است زمان بیشتری نیاز داشته باشند
- **اینترنت:** نیاز به اتصال اینترنت برای Google Speech Recognition
